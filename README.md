# 자동차 경주 게임

## 진행 방법

1. 경주에 참여 할 자동차 입력을 입력 한다.(,로 구분)
2. 경주가 몇 라운드로 진행 될지 입력 한다.
3. 한 라운드 당 한 차량은 전진 또는 멈출 수 있다.

- 전진 조건 : 0~9의 랜덤 숫자 중 0~3이 나온 경우
- 멈춤 조건 : 0~9의 랜덤 숫자 중 4~0가 나온 경우

4. 모든 라운드가 진행 된 후, 최종 우승자를 알 수 있다

- 우승자는 한 명 이상 일 수 있다.
